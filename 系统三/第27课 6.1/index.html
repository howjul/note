    FLynnтѕєу▒╗Т│Ћ¶

    FlynnС║ј1972т╣┤ТЈљтЄ║С║єУ«Ау«Ќт╣│тЈ░уџёFlynnтѕєу▒╗Т│Ћ№╝їСИ╗УдЂТа╣ТЇ«ТїЄС╗цТхЂтњїТЋ░ТЇ«ТхЂТЮЦтѕєу▒╗№╝їтЁ▒тѕєСИ║тЏЏуДЇу▒╗тъІуџёУ«Ау«Ќт╣│тЈ░№╝їSт░▒Тў»тЇЋ№╝їMт░▒Тў»тцџ№╝їIт░▒Тў»ТїЄС╗ц№╝їDт░▒Тў»ТЋ░ТЇ«ТхЂсђѓ

    тЇЋТїЄС╗цТхЂтЇЋТЋ░ТЇ«ТхЂТю║тЎе№╝ѕSISD№╝Ѕ№╝џSISDТю║тЎеТў»СИђуДЇС╝ау╗ЪуџёСИ▓УАїУ«Ау«ЌТю║№╝їт«ЃуџёуАгС╗ХСИЇТћ»ТїЂС╗╗СйЋтйбт╝Јуџёт╣ХУАїУ«Ау«Ќ№╝їТЅђТюЅуџёТїЄС╗цжЃйТў»СИ▓УАїТЅДУАїсђѓт╣ХСИћтюеТЪљСИфТЌХжњЪтЉеТюЪтєЁ№╝їCPUтЈфУЃйтцёуљєСИђСИфТЋ░ТЇ«ТхЂсђѓтЏаТГцУ┐ЎуДЇТю║тЎеУбФуД░СйютЇЋТїЄС╗цТхЂтЇЋТЋ░ТЇ«ТхЂТю║тЎесђѓТЌЕТюЪуџёУ«Ау«ЌТю║жЃйТў»SISDТю║тЎе№╝їтдѓтє»У»║СЙЮТЏ╝ТъХТъёсђѓ

    тЇЋТїЄС╗цТхЂтцџТЋ░ТЇ«ТхЂТю║тЎе№╝ѕSIMD)

    SIMDТў»жЄЄућеСИђСИфТїЄС╗цТхЂтцёуљєтцџСИфТЋ░ТЇ«ТхЂсђѓУ┐Ўу▒╗Тю║тЎетюеТЋ░тГЌС┐АтЈитцёуљєсђЂтЏЙтЃЈтцёуљєсђЂС╗ЦтЈітцџтфњСйЊС┐АТЂ»тцёуљєуГЅжбєтЪЪжЮътИИТюЅТЋѕIntelтцёуљєтЎет«ъуј░уџёMMXTMсђЂSSE№╝ѕStreaming SIMD Extensions№╝ЅсђЂSSE2тЈіSSE3ТЅЕт▒ЋТїЄС╗цжЏє№╝їжЃйУЃйтюетЇЋСИфТЌХжњЪтЉеТюЪтєЁтцёуљєтцџСИфТЋ░ТЇ«тЇЋтЁЃсђѓС╣Ът░▒Тў»У»┤ТѕЉС╗гуј░тюеућеуџётЇЋТаИУ«Ау«ЌТю║тЪ║ТюгСИіжЃйт▒ъС║јSIMDТю║тЎесђѓ

    тцџТїЄС╗цТхЂтЇЋТЋ░ТЇ«ТхЂТю║тЎе№╝ѕMISD№╝Ѕ

    MISDТў»жЄЄућетцџСИфТїЄС╗цТхЂТЮЦтцёуљєтЇЋСИфТЋ░ТЇ«ТхЂсђѓућ▒С║јт«ъжЎЁТЃЁтєхСИГ№╝їжЄЄућетцџТїЄС╗цТхЂтцёуљєтцџТЋ░ТЇ«ТхЂТЅЇТў»ТЏ┤ТюЅТЋѕуџёТќ╣Т│Ћ№╝їтЏаТГцMISDтЈфТў»СйюСИ║уљєУ«║ТеАтъІтЄ║уј░№╝їТ▓АТюЅТіЋтЁЦтѕ░т«ъжЎЁт║ћућеС╣ІСИГсђѓ

    тцџТїЄС╗цТхЂтцџТЋ░ТЇ«ТхЂТю║тЎе№╝ѕMIMD№╝Ѕ

    MIMDТю║тЎетЈ»С╗ЦтљїТЌХТЅДУАїтцџСИфТїЄС╗цТхЂ№╝їУ┐ЎС║ЏТїЄС╗цТхЂтѕєтѕФт»╣СИЇтљїТЋ░ТЇ«ТхЂУ┐ЏУАїТЊЇСйюсђѓТюђТќ░уџётцџТаИУ«Ау«Ќт╣│тЈ░т░▒т▒ъС║јMIMDуџёУїЃуЋ┤№╝їСЙІтдѓIntelтњїAMDуџётЈїТаИтцёуљєтЎеуГЅжЃйт▒ъС║јMIMDсђѓ

    Vector Processor & Scalar Processor¶

    ТаЄжЄЈтцёуљєтЎе№╝џТаЄжЄЈтцёуљєтЎеСИђТгАтЈфтцёуљєСИђСИфТЋ░ТЇ«№╝їтИИУДЂуџёТЋ░ТЇ«у▒╗тъІТюЅТЋ┤тъІсђЂТх«уѓ╣тъІуГЅсђѓТаЄжЄЈтцёуљєтЎет▒ъС║јтЇЋТїЄС╗цтЇЋТЋ░ТЇ«№╝ѕSISD№╝ЅСИђу▒╗сђѓ

    тљЉжЄЈтцёуљєтЎе№╝џтљЉжЄЈтцёуљєтЎеСИГ№╝їСИђТЮАТїЄС╗цтљїТЌХТЊЇСйютцџСИфТЋ░ТЇ«№╝ѕSIMD№╝Ѕсђѓ

    ТхЂТ░┤у║┐У┐ЏУАїтљЉжЄЈУ┐љу«Ќ

    СЙІтГљ№╝џУдЂТ▒ѓ\(D = A \times (B + C)\)№╝ї\(A№╝їB№╝їC№╝їD\)Тў»жЋ┐т║дСИ║NуџётљЉжЄЈсђѓ

    • 1№╝ЅHorizontal processing method

    Т░┤т╣│тіатиЦТќ╣Т│Ћ№╝їуЪбжЄЈУ«Ау«ЌТў»тюеСИђУАїСИГС╗јтидтѕ░тЈ│Т░┤т╣│ТЅДУАїуџёсђѓТюЅ2NТгАуџётіЪУЃйтѕЄТЇбсђѓ

    • 2№╝Ѕ Vertical processing method

    тъѓуЏ┤тіатиЦТќ╣Т│Ћ№╝їуЪбжЄЈУ«Ау«ЌТў»С╗ЦтѕЌуџёТќ╣т╝ЈС╗јСИітѕ░СИІтъѓуЏ┤ТЅДУАїуџёсђѓтдѓТъюжЋ┐т║дNтЙѕтцД№╝їСИђТгАТђДТЌаТ│ЋТћЙСИІN№╝їжѓБС╣ѕС╝џтЄ║уј░жЌ«жбўсђѓ

    • 3№╝Ѕ Vertical and horizontal processing method (group processing method)

    у║хТефтіатиЦТ│Ћ№╝ѕТѕљу╗ётіатиЦТ│Ћ№╝Ѕ№╝їТ░┤т╣│тцёуљєтњїтъѓуЏ┤тцёуљєуџёу╗Њтљѕ№╝їТіітљЉжЄЈтѕєТѕљУІЦт╣▓у╗ё№╝їтѕєтѕФУ┐ЏУАїтъѓуЏ┤тіатиЦу«ЌТ│Ћсђѓ

    ТѕЉС╗гТііжЋ┐т║дNтѕєТѕљS+1у╗ё№╝їТ»Ју╗ёnСИфтЁЃу┤а№╝їу╗ётєЁу║хтљЉ№╝їу╗ёжЌ┤ТефтљЉ
